# 🎨 Portfolio Management - Creator Workspace

A sleek, futuristic CMS dashboard for creators to manage their professional presence. Built with Vue 3 and designed with a clean, minimal aesthetic.

## 🌟 Key Features

- **✍️ Blog Editor**: Full-featured rich text editor for writing and publishing articles.
- **🤖 AI Writing Assistant**: Integrated AI panel to help generate outlines, rewrite drafts, and craft compelling titles.
- **💼 Portfolio Manager**: Showcase your projects with multi-language support (ID/EN) and tech stack management.
- **📄 CV/Resume Upload**: Manage your professional documents with a dedicated upload area and primary document selector.
- **⚙️ Profile Settings**: Configure your custom username which acts as your public API identifier.
- **📟 Developer API Docs**: A dedicated documentation page with interactive snippets (cURL, Fetch) to help you integrate your data into your own site.

## 🛠️ Tech Stack

- **Framework**: Vue 3 (Composition API)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: Pinia
- **Components**: Custom minimal components + Vue Quill for editing.
- **Icons/Visuals**: Inline SVG + futuristic glassmorphism effects.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Setup Environment**:
    Create a `.env` file:
    ```env
    VITE_API_BASE_URL=http://localhost:3000
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## 📐 Design Philosophy

- **Clean & Minimal**: Focus on content with a soft neutral palette.
- **Futuristic Avenues**: Subtle gradients, pulse animations, and glassmorphism.
- **Developer-Centric**: The API Docs page uses a "dark fantasy" theme with glitch effects for a high-tech developer experience.
